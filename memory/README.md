# tracking — Daily Operations

This folder is the operational heartbeat of RIGGS. The morning check-in writes to it. The evening check-in writes to it. Over time it becomes a complete record of your days.

---

## Files in This Folder

| File | Purpose |
|------|---------|
| `daily_log.md` | Daily entries — journal, foundational tasks, score, notes |
| `goals/weekly.md` | This week's goals and task list |
| `goals/monthly_[YYYY-MM].md` | Monthly goals and progress |

---

## How to Use This Folder

- Do not manually edit `daily_log.md` during a session — let RIGGS write to it during check-ins
- Review weekly goals every Monday morning and update them
- Monthly goal files are created by RIGGS at the start of each month
- The daily score (if you use the scoring rubric) is logged here and feeds into weekly/monthly summaries
