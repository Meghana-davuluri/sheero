# templates — Templates and Frameworks

This folder holds reusable templates for documents, communications, and prompts you produce repeatedly. Store a template here once and reference it every time instead of writing from scratch.

---

## What Belongs Here

- Email templates (outreach, follow-up, cover letters)
- Document templates (meeting notes, project briefs, reports)
- Prompt frameworks (structured prompts for recurring AI tasks)
- Writing frameworks (blog post structure, X thread format, etc.)

---

## How to Use This Folder

- Name files descriptively: `email_cold_outreach.md`, `project_brief.md`, `weekly_review_prompt.md`
- Include placeholders in `[BRACKETS]` for variable content
- Sheero can reference these when you ask it to draft something — point it to the relevant template
